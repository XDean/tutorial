## 关于React

React是一个用于构建用户界面的Javascript库。
React是当下最流行的前端框架之一。

## 关于本文档

本文档主要参考[官方文档](https://zh-hans.reactjs.org/docs/getting-started.html)，并提供交互式代码编辑器帮助学习。

要阅读本文档，你需要首先了解基本的HTML，JS，CSS知识。
