我们似乎直接在JS脚本中写了HTML。
然而这些代码既不是HTML也不是字符串。

这其实是JSX拓展语法。它允许我们在JS代码中写入类似于HTML的代码，实际上它们会被转换成React元素对象。例如：

```jsx
const App = () => (
  <h1 className="greeting">
    Hello, world!
  </h1>
);

//会变成

const App = () => React.createElement(
  'h1',
  {className: 'greeting'},
  'Hello, world!'
);
```

关于React元素对象的细节将会在下一节讲到。

有了JSX，我们可以方便的在JS里写UI代码。同时，因为本质还是JS，我们可以在JSX里插入JS代码段。
只需要用大括号`{}`框住，可以在属性或是内容里插入JS，事件处理函数也可以直接插入其中。
