上一节讲到，我们可以在JSX中用便捷的标签语法定义React元素。

React元素并不是浏览器的DOM元素，而是Javascript对象，创建它们的开销远小于DOM元素。
而React的工作就是把React元素渲染成DOM元素。
并且在此过程中，React会比较元素的diff，并进行最小化的更新，而非重建整个DOM。
