在基础篇我们学习了React本体，通过这些知识，我们可以运用React创建任何网页应用。

## 工具链

但是在实际工程中，我们一般不会直接裸写Javascript脚本。
因为Javascript脚本难以组织，难以依赖三方库，语言特性不统一，有着诸多问题和限制。

所以我们有一系列的工具链来帮助我们开发现代化前端应用。它们可以帮助我们

- 扩展文件和组件的规模，有效组织大型项目
- 便捷地依赖第三方库
- 尽早发现错误(例如类型检查)
- 本地开发服务器，编辑后就能够实时预览
- 优化输出的JS脚本(例如文件大小)

本篇教程会介绍一些主流的工具链，以及两个主流的React框架

- Create React App
- NextJS

需要注意的是，虽然这里是React教程，但是React核心部分已经结束。
接下来学习的技术和工具链大多与React不是强相关，不使用它们你也可以编写React应用，正如我们在基础篇所学的那样。

## 类库

另一方面，我们的项目一定不是从零开始创造世界，所谓君子善假于物也，我们会在各个功能上依赖第三方库。

本篇教程会对各个常用的功能模块介绍相关的流行类库，以帮助你快速了解它们。
